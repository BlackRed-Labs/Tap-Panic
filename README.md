# Tap Panic

A fast-paced arcade reflex game built in Unity where players must survive the chaos, maintain control, and chase high scores.

## 🎮 About The Game

**Tap Panic** is a rhythm-based arcade survival game focused on quick reactions, smooth gameplay, and increasing difficulty. The gameplay starts simple but becomes more intense as the speed, movement, and pressure increase.

Players must:

* Tap on the ball
* Avoid losing health
* Keep the balls under control
* Survive longer to achieve higher scores
* Adapt to increasing difficulty and faster gameplay

The game is designed with satisfying bounce physics, smooth UI animations and energetic gameplay.

# 🧠 Gameplay Loop

1. Balls spawn into the scene
2. Player taps and reacts quickly
3. Difficulty increases over time
4. Mistakes reduce health
5. Survive as long as possible
6. Earn coins and beat high scores
